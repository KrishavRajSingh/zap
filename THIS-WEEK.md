Use this file if you feel overwhelmed.

Your job this week is simple:

- understand the buyer
- make Zap easier to explain
- start outreach
- choose one external OSS repo
- do not get distracted

By the end of this week, you should have:

- read the beginner files
- created your CRM
- added the first 10 target companies
- sent the first 5 outreach messages
- practiced the call script out loud
- chosen one external OSS repo

That is a good week.
